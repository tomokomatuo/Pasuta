# README

自分に似合う洋服のスタイリングがわからない人、自分の服のセンスに自信がない人のためにパーソナルスタイリスト（服アドバイザー）を派遣するアプリ。

## バージョン情報
* Ruby 2.6.5
* Ruby on Rails 5.2.4
* psql (PostgreSQL) 12.2
